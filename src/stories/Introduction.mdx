# Flux Design System

A collection of reusable components and design foundations built to create consistent interfaces across products and brands.

## Overview

Use this documentation to explore the available components, and themes.

Each component includes documentation, examples, and interactive controls to help understand its usage and available options.

## Components

Browse the component library through the sidebar. Each component includes:

- Documentation and usage examples
- Available variants and properties
- Interactive previews
- Theme support

## Themes

Flux supports multiple themes that can be previewed directly within Storybook.

Use the theme selector in the toolbar to switch between available themes and view how components adapt across different brand styles.
